# 🚀 Full Stack Development Lab (Sem 6)

## 📌 Overview

This repository presents a series of 10 structured full-stack development experiments. It covers frontend development using React, backend development using Spring Boot, and integration of both using REST APIs, JWT authentication, and basic real-time communication.

Each experiment builds progressively to provide a clear understanding of modern full-stack application development.

---

## 🛠️ Tech Stack

* React.js
* React Router
* Context API & Redux Toolkit
* Bootstrap & Material UI
* Java (Spring Boot)
* REST API
* JWT Authentication
* WebSockets

---

## 📂 Experiment Breakdown

### 🔹 Exp01 – React Basics & SPA

Introduction to React concepts including components and state management using `useState`. Includes simple applications like a ToDo List.

### 🔹 Exp02 – UI Development

Designing responsive user interfaces using Bootstrap and Material UI.

### 🔹 Exp03 – Routing

Implementing navigation using React Router with support for multiple pages and dynamic routing.

### 🔹 Exp04 – State Management

Understanding and implementing global state using Context API and Redux Toolkit.

### 🔹 Exp05 – Performance Optimization

Improving application performance using lazy loading and code splitting techniques.

### 🔹 Exp06 – React Hooks

Using hooks such as `useEffect` and `useMemo` to handle lifecycle and optimize rendering.

### 🔹 Exp07 – Context API

Managing shared data efficiently and avoiding prop drilling in component trees.

### 🔹 Exp08 – Spring Boot REST API

Developing backend services using Spring Boot with basic CRUD operations and REST architecture.

### 🔹 Exp09 – JWT Authentication

Implementing secure authentication using JSON Web Tokens for login and protected routes.

### 🔹 Exp10 – Full Stack Integration

Integrating React frontend with Spring Boot backend using REST APIs, along with basic real-time updates using WebSockets.
